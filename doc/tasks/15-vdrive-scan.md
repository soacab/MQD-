# 15 VDrive 文件夹扫描

## 目标
接入 VDrive 文件夹扫描，保存文件夹和文件快照，为文件存在性自动检查提供输入。

## 参考文档
- `CheckFlow MVP 开发任务拆分 v1 0 38e75a5befd780778d1ef8bdd4cd1e65.md`：Phase 6。
- `CheckFlow 系统架构设计 v1.md`：VDrive Adapter。
- `CheckFlow 数据库设计 v1 0.md`：VDrive 快照表。

## 前置依赖
- [ ] 04 VDrive 链接校验完成。
- [ ] 08 生成轮次和检查项完成。

## 最小任务清单
- [ ] T01：实现 `POST /api/v1/projects/{project_id}/vdrive/scan`；验收：可创建扫描批次。
- [ ] T02：实现 VDrive 文件夹扫描 Adapter；验收：可读取根目录文件和子文件夹。
- [ ] T03：保存 `vdrive_scan_batches`；验收：记录项目、状态、开始时间、结束时间。
- [ ] T04：保存 `vdrive_folders` 和 `vdrive_files`；验收：文件路径、名称、版本、时间、大小可查询。
- [ ] T05：实现扫描结果查询接口；验收：可按批次查看扫描状态和文件数量。
- [ ] T06：处理扫描失败；验收：失败原因入库并可在前端展示。
- [ ] T07：实现前端扫描触发和结果展示；验收：用户可看到扫描中、成功、失败状态。

## 验收标准
- [ ] 可以扫描 VDrive 文件夹。
- [ ] 可以保存文件夹和文件快照。
- [ ] 扫描失败不影响项目历史数据。
- [ ] 扫描批次可被自动检查模块引用。

## 注意事项
MVP 可先做非递归扫描；递归扫描在 Adapter 接口稳定后再打开。
