## 验收标准
- [ ] 可以在前端完整创建、查询、编辑项目，并覆盖方案要求的项目基础字段。
- [x] 可以保存多个项目接收批次和机型。
- [x] 删除项目后普通列表不展示。
- [x] `deleted` 项目不能继续创建点检任务。
- [ ] 删除项目前端存在真实二次确认，且必须手动输入项目名称。
- [ ] VDrive 校验后支持历史信息回填和 QG 节点推荐。
- [ ] 加单遵守无进行中任务、无复查中任务的业务前置条件。

## 注意事项
