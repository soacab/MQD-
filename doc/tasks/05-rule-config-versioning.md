## 注意事项
业务规则用于展示和生成检查项；自动执行规则用于后端真实执行，两者不能混为一张表。

方案设计中还包含人工检查项升级建议等交互细节；这些不影响当前 P0 后端主链路。当前人工检查项“删除”按停用 `is_active=false` 语义落地，不做物理删除。
