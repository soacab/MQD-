- [x] 已发布版本可用于创建点检任务。

## 注意事项
业务规则用于展示和生成检查项；自动执行规则用于后端真实执行，两者不能混为一张表。

方案设计中还包含人工检查项升级建议、人工检查项删除、发布弹窗等交互细节；这些不影响当前 P0 后端主链路，但在补齐规则配置前端时需要一起复核优先级。
