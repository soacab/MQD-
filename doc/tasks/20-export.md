# 20 导出能力（P2 暂缓）

## 目标
支持检查档案 Excel 导出和节点报告 PDF 导出，满足历史追溯和对外沟通需要。

## 前置依赖
- [ ] 11 报告生成完成。
- [ ] 17 工作台与查询完成。

## 最小任务清单
- [x] T00：前端检查档案页提供临时 Excel 下载；验收：可把当前页面数据拼成表格文件下载。该能力不包含后端导出接口、异步任务、对象存储或导出审计。
- [ ] T01：实现档案列表 Excel 导出接口；验收：导出字段与当前筛选列表一致。
- [ ] T02：实现报告详情 PDF 导出接口；验收：包含项目明细、综合结论、检查项明细和过程记录。
- [ ] T03：设计导出任务状态；验收：大文件导出可异步查询进度。
- [ ] T04：接入对象存储保存导出文件；验收：导出完成后返回可下载地址。
- [ ] T05：实现前端导出按钮和状态提示；验收：用户可触发导出、查看进度、下载结果。
- [ ] T06：记录导出审计日志；验收：导出人、时间、筛选条件可追溯。

## 验收标准
- [ ] Excel 导出遵循当前筛选条件。
- [ ] PDF 报告内容与页面展示一致。
- [ ] 导出失败有明确错误提示。

## 注意事项
本模块标记为 P2 暂缓；P0 主流程完成前不要投入复杂版式优化。
当前已有的“导出 Excel”只是前端临时下载能力，不代表本模块的正式导出能力已启动。
